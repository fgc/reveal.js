Backtracking(*)

Backtracking(*)?

  • Cada unificación devuelve una lista perezosa de sustituciones que unifican.
  • Búsqueda exhaustiva que concatena todas las sustituciones válidas en una lista final. Perezosa por supuesto.
  • El resultado final es equivalente al de hacer backtracking "normal".

    • * Una pega, JavaScript no tiene listas perezosas...

    ... Pero el propio SICP explica como implementarlas en el capítulo 3.

Global State

Set data-state="something" on a slide and "something" will be added as a class to the document element when the slide is open. This lets you apply broader style changes, like switching the background.

Custom Events

Additionally custom events can be triggered on a per slide basis by binding to the data-state name. 


	      Reveal.addEventListener( 'customevent', function() {
	      console.log( '"customevent" has fired' );
	      } );
